# Mensa Griebnitzsee Discord Bot

A GitHub Actions-based bot that automatically posts the daily menu from Mensa Griebnitzsee (Uni Potsdam) to Discord. 

This repo is intentended for private use, but it could be easily extended so that you can use use it with any mensa that is available on www.imensa.de. So if you are interested, let me know!
